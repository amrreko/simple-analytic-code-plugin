== Description ==

A simple code to Google Analytics, no more load, no more code.
Google Analytics code in your WordPress header or footer as you like.
Inserts the tracking code in every page of your site.

== Installation ==
Just uplode small plugin and active it.
Edit plugin and set your Universal Analytics key UA-xxxxxxxx-x.
Get from https://www.google.com/analytics you must register your website to get
